# CineScope

A movie and TV series discovery app with a Stranger Things-inspired vibe.

## What it does

- Search for movies and series using the OMDb API
- Filter by genre (Crime, Sci-Fi, Comedy, Adventure)
- Sort by year, title, or relevance
- Save favorites locally (stored in IndexedDB)
- Hover over posters to preview trailers from YouTube
- View detailed info including cast, director, ratings, and box office

## Tech

- Vanilla HTML, CSS, JavaScript
- OMDb API for movie data
- YouTube Data API for trailer previews
- IndexedDB for storing favorites
- Custom Stranger Things-style fonts

## Run it

Just open `index.html` in your browser. No build step needed.

## Preview

The app opens with a cinematic intro screen, then drops you into a curated homepage of popular movies. Search for anything, click a card for details, and heart your favorites.
